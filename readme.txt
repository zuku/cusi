Cusi is command line tools for M5Stack MicroPython (UIFlow) system.

This archive contains command files for several platforms.

* darwin_amd64/cusi
  * for Intel Mac
* darwin_arm64/cusi
  * for Apple Silicon Mac
* linux_amd64/cusi
  * for Linux (x86_64)
* windows_amd64/cusi.exe
  * for Windows (x86_64)

For more information, please visit https://github.com/zuku/cusi
